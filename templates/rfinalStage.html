The regression results displayed above are based on the following models: Linear Regression, Ridge Regression, Lasso Regression, Support Vector Regressor, Random Forest Regressor, and Gradient Boosting Regressor.

Each model has been tuned using grid search over the following hyperparameter ranges:

  • Linear Regression: No hyperparameters to tune.
  • Ridge Regression: Regularization strength (alpha = 0.1, 1, 10, 100).
  • Lasso Regression: Regularization strength (alpha = 0.1, 1, 10, 100).
  • Support Vector Regressor: Regularization strength (C), kernel type (linear, rbf), and gamma (scale, auto).
  • Random Forest Regressor: Number of estimators (50, 100, 200) and maximum depth (None, 10, 20).
  • Gradient Boosting Regressor: Number of estimators (50, 100), learning rate (0.01, 0.1), and maximum depth (3, 5, 7).

The best model is selected based on its performance on the test dataset using metrics like R², MSE, and MAE. Please note that the results depend on the dataset and hyperparameter grid provided for this analysis.
