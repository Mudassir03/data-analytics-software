Disclaimer

The results displayed above are based on multiple machine learning models, including Logistic Regression, Support Vector Classifier, Decision Tree, Random Forest, and Gradient Boosting Classifier.

Each model has been optimized using a grid search over the following hyperparameters:

  • Logistic Regression: Regularization strength (C), penalty (l1, l2), and solver (liblinear).
  • Support Vector Classifier: Regularization strength (C), kernel type (linear, rbf), and gamma (scale, auto).
  • Decision Tree: Maximum depth, and minimum samples for a split.
  • Random Forest: Number of estimators, maximum depth, and minimum samples for a split.
  • Gradient Boosting Classifier: Number of estimators, learning rate, and maximum depth.

The final model selection is based on the best performance on the testing data. Please note that the results are influenced by the dataset and hyperparameter search space provided.
